# Niangaly Abdoulaye

Développeur full-stack basé à Bamako, Mali.

Je construis des applications web, des APIs, des SDKs et des outils d'automatisation, avec un focus sur les produits utiles, les intégrations paiement/SMS, les plateformes Laravel et les expériences mobiles.

## Ce que je fais

- Applications web avec PHP, Laravel, Inertia, React, Blade et Tailwind CSS
- APIs et SDKs avec TypeScript, Node.js et architectures propres
- Intégrations paiement, SMS, automatisation et services tiers
- Applications mobiles avec Flutter et Dart
- Projets autour de l'IA, de la recherche et des outils productivité
